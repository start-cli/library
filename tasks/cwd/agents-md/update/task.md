# Update AGENTS.md

Update an existing AGENTS.md document for a repository. A README for AI agents with context and instructions for repository work.

## Instructions

1. Fetch the AGENTS.md specification from https://agents.md/ for context and format
2. Confirm an AGENTS.md file exists in the repository root
3. Follow the process below to update it

## Process

Update an existing AGENTS.md by:

1. Read the current AGENTS.md to understand its structure
2. Analyse the repository for changes since last update
3. Identify outdated, missing, or incorrect information
4. Update sections based on the user's request or discovered changes

Preserve any repository-specific customisations.

Targeted: focus on requested section. Refresh: review entire document against current state.

## Restrictions

When writing the AGENTS.md document, do not use:

- Bold or italic formatting
- Horizontal rules
- Emojis
- HTML comments
- Image embeds (use regular links)
- Multiple consecutive blank lines
- Nested lists beyond 3 levels
- Task lists
- Heading depth beyond `###`
- Directory structures beyond depth 3

Keep:

- Headings for structure
- Single blank lines between sections
- Inline code and code blocks
- Tables for structured data
- Lists (ordered and unordered)
- Callout prefixes without bold

## Style Guidelines

- Use a descriptive project title as the main heading, not "AGENTS.md"
- Avoid repeating information across sections; consolidate or cross-reference
- Keep table columns consistent in purpose
