# Deep Research Jira Work Item

Investigate a Jira work item across every available data source and compile a structured research report.

## Instructions

1. Read the work item and all comments
2. Search other Jira projects for related or similar items
3. Clone and analyse the related git repository if identifiable
4. Search documentation (Confluence, wikis, internal docs)
5. Analyse the cloud environment if relevant
6. Analyse Kubernetes resources if relevant
7. Compile a structured research report with findings per source
