# Review Jira Work Item

Perform a comprehensive content review of a single Jira work item.

## Instructions

1. Read the work item description and last 5 comments
2. Request more comments if the discussion indicates there is more
3. Evaluate description accuracy, detail level, and completeness
4. Check for mistakes, inconsistencies, or contradictions
5. Identify open questions or ambiguities
6. Light external research to validate claims or assumptions if needed
7. Produce a review report with findings and suggestions
