# Review Jira Blocked Items

Review the user's blocked items to find resolution paths.

## Instructions

1. Query Jira for the user's blocked items
2. For each item: read the description, comments, linked items, and blockers
3. Research whether the blocker is resolved or has a workaround
4. Assess whether the item should remain blocked, move forward, or escalate
5. Produce a summary report with status and recommended actions per item
