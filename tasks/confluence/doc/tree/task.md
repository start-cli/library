# Read Confluence Document Tree

Read a Confluence document and its child pages, providing a hierarchical summary.

## Instructions

1. Fetch the root Confluence document using available tools
2. Retrieve the document's child pages (direct descendants)
3. Analyse the document hierarchy and content
4. Respond with a structured summary covering:
   - Root document title and purpose
   - List of child pages with brief descriptions
   - Overall structure and organisation of the content
   - Key topics covered across the document tree
